13.

Uvažte následující React aplikaci:

import React, { useState } from 'react';
import { render } from 'react-dom';

const App = () => {
  const [active, setActive] = useState(true);

  return (
    <button
      className="btn"
      onClick={() => setActive(!active)}
    >
      Ok
    </button>
  );
};

render(<App />, document.getElementById('app'));

Aplikace při každém kliknutí na tlačítko mění hodnotu stavu active z true na false a obráceně.

Chceme, aby se tato změna projevila vizuálně i na tlačítku. Předpokládejme, že v CSS máme nastylovanou třídu btn--active. Když tuto třídu přidáme na tlačítko, bude tlačítko zvýrazněné jako aktivní.

Upravte aplikaci uvedenou výše tak, aby se na tlačítku objevila CSS třída btn--active, je-li stav active nastaven na true. Je-li ve stavu active hodnota false, třída na tlačítku nebude.

Jako řešení stačí poslat kód komponenty App.
