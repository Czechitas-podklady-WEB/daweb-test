13.

Uvažte následující Vue.js aplikaci:

<div id="app">
  <button
    class="btn"
    v-on:click="btnClick"
  >
  Ok
  </button>
</div>
<script src="vue.js"></script>
<script>
  new Vue({
    el: '#app',
    data: {
      active: true,
    },
    methods: {
      btnClick() {
        this.active = !this.active;
      }
    }
  });
</script>

Aplikace při každém kliknutí na tlačítko mění hodnotu vlastnosti active z true na false a obráceně.

Chceme, aby se tato změna projevila vizuálně i na tlačítku. Předpokládejme, že v CSS máme nastylovanou třídu btn-active. Když tuto třídu přidáme na tlačítko, bude tlačítko zvýrazněné jako aktivní.

Upravte aplikaci uvedenou výše tak, aby se na tlačítku objevila CSS třída btn-active, je-li hodnota vlastnosti active nastavena na true. Je-li hodnota vlastnosti active nastavena na false, třída na tlačítku nebude.

Jako řešení stačí poslat pouze řádky se změněným kódem. Mělo by se jednat pouze o 1-3 řádky.
